Please set the following environment variables before running the application.

Step 1: Set Environment Variables

Create a .env file or export these variables:

export SHAREPOINT_SITE_URL="https://yourcompany.sharepoint.com/sites/yoursite" export SHAREPOINT_USERNAME="your.email@company.com" export SHAREPOINT_PASSWORD="your-password"

Step 2: Restart the Server

After setting the environment variables, restart the Flask application:

python app.py

Alternative: Use .env File

You can also create a .env file in the project root with:

SHAREPOINT_SITE_URL=https://yourcompany.sharepoint.com/sites/yoursite SHAREPOINT_USERNAME=your.email@company.com SHAREPOINT_PASSWORD=your-password

Then install python-dotenv and load it in app.py:

pip install python-dotenv
